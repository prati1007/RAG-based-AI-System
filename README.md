# RAG-based AI Assistant for Edge Deployment

## 📌 Overview
This project builds a Retrieval-Augmented Generation (RAG) system to recommend AI model optimization strategies based on hardware constraints.

## 🚀 Features
- Domain-specific document retrieval using FAISS
- LangChain-based agent workflows
- Context-aware LLM responses

## 🏗️ Pipeline
User Query → Embedding → FAISS Retrieval → LLM → Response

## ⚙️ Tech Stack
- Python, LangChain, FAISS
- OpenAI / LLM APIs

## 📊 Applications
- Edge deployment recommendation
- Model selection under constraints
